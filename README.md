# PCB Drill File Optimizer

A Python tool to parse Excellon (.drl) PCB drill files and generate optimized G-code.
This project is being developed as preparation for Google Summer of Code 2026 under PSF.

## Planned Features
- Drill file parser
- G-code generator
- Tkinter GUI
- CLI tool
- Drill preview visualization

  
## Development Status

This project is under active development as part of Google Summer of Code 2026
preparation under the Python Software Foundation.
